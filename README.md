# 🚀 Zerodha Dashboard (Frontend Clone)

A modern Zerodha Dashboard UI clone built using **React.js**, designed to simulate a stock trading interface with real-time-like data visualization using static datasets.

## 🌐 Live Demo

👉 [https://zerodha-dashboard-j9me.vercel.app/](https://zerodha-dashboard-j9me.vercel.app/)

---

## 📌 Features

- 📊 Interactive Dashboard UI
- 📈 Doughnut Chart for stock distribution
- 📃 Watchlist with dynamic price indicators
- 💹 Holdings & Positions table
- 🎯 Clean and responsive UI
- ⚡ Fast performance (no backend dependency)

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| Frontend | React.js |
| Styling | CSS / Bootstrap / MUI |
| Charts | Chart.js + react-chartjs-2 |
| Routing | React Router |
| State Management | React Context API |

## ⚙️ Installation & Setup

**Clone the repository:**

```bash
git clone https://github.com/kumar-veerendra/zerodha-dashboard.git
cd zerodha-dashboard
```

**Install dependencies:**

```bash
npm install
```

**Run the app:**

```bash
npm start
```

---

## 🚀 Deployment

Deployed using **Vercel**

**Steps:**
1. Push code to GitHub
2. Import project in Vercel
3. Deploy *(Auto configuration for React)*
